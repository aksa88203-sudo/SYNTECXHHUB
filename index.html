How A* works
A* (A-star) is an informed search algorithm. It evaluates each node by combining g(n) — the actual cost from start to node — and h(n) — the heuristic estimate to the goal: f(n) = g(n) + h(n).

Manhattan heuristic: |Δrow| + |Δcol| — optimal for grid movement with 4 directions.
Euclidean heuristic: √(Δrow² + Δcol²) — straight-line distance, admissible but may explore more cells.

A* guarantees the shortest path as long as the heuristic never overestimates the true cost (admissibility).
